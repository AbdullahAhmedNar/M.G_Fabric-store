ملاحظة عن الأيقونة:

الملف الحالي icon.png هو SVG مضمّن.

لبناء ملف EXE نهائي، يجب تحويل هذه الأيقونة إلى صيغة ICO:

1. اذهب إلى موقع: https://convertico.com/
2. ارفع ملف icon.png
3. حوّله إلى icon.ico بحجم 256x256
4. ضع الملف الناتج في مجلد assets/

أو استخدم أيقونة جاهزة من:
- https://www.flaticon.com/
- https://icons8.com/

الأيقونة المقترحة: 
رمز قماش أو بكرة خيوط أو مقص خياطة بألوان البرنامج (جملي/بني)
